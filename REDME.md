# prismFX ARGB Controller (PC Host)

prismFX ARGB Controller 上位机软件，用于与 prismFX ARGB 控制器下位机通信与配置。

本项目需配合下位机仓库使用

## 📖 简介
- 🐍 Python + Qt (PySide6) 开发
- 🎨 UI 使用 QFluentWidgets
- 🌈 用于 ARGB 灯效控制与设备配置
- 🔗 与 prismFX ARGB Controller Firmware 配套
- 📡 BLE 通信架构，支持后续通信扩展

## 🔗 下位机仓库
